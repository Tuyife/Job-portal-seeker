#  Job Board Portal

A fully functional job board web application built with React and Vite. Users can browse jobs, search/filter listings, bookmark favorites, view detailed job information, and submit job applications.

##  Features

-  Search & Filter** - Search jobs by title, company, or location. Filter by category
-  Bookmark System** - Save favorite jobs for later viewing
-  Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
-  Application Form** - Submit job applications with form validation
-  Local Storage** - Bookmarks and applications are saved in your browser
- Fast & Modern** - Built with Vite for lightning-fast development and builds

##  Technologies Used

- **React 18** - Frontend framework
- **Vite** - Build tool and development server
- **CSS3** - Styling with modern flexbox/grid layouts
- **LocalStorage API** - Persistent data storage
